## 🎬 ANIMAÇÕES DISPONÍVEIS

### SCROLL ANIMATIONS:

```
1. scroll-fade-up      [Elemento sobe com fade in]
2. scroll-fade-down    [Elemento desce com fade in]
3. scroll-slide-left   [Desliza da esquerda]
4. scroll-slide-right  [Desliza da direita]
5. scroll-scale        [Amplia com zoom]
6. scroll-rotate       [Rotaciona ao entrar]

Tempo: 0.8s por padrão
Trigger: Quando elemento aparece na tela
```

### CONTINUOUS EFFECTS:

```
7. glow-effect         [Brilho rosa pulsante infinito]
8. pulse-effect        [Pulso de escala infinito]
9. float-effect        [Flutuação suave infinita]
10. border-glow        [Borda com brilho infinito]
```

### HOVER EFFECTS:

```
Botões         → Levanta 5px + glow rosa
Cards          → Move para cima + sombra
Ícones         → Flutuam e expandem
Links          → Linha animada embaixo
Scrollbar      → Gradiente mais vibrante
```

## 🔧 ARQUITETURA TÉCNICA

### CSS Variables (Dark Theme):
```css
:root {
    --bg-primary: #0f0f1e;
    --bg-secondary: #1a1a2e;
    --text-primary: #e4e4e7;
    --accent-primary: #d946a6;
    --gradient-primary: linear-gradient(135deg, #d946a6, #ec4899);
    /* ... mais 10 variáveis */
}
```

### Intersection Observer (Scroll Detection):
```javascript
const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            // Trigger animation
            entry.target.classList.add('active');
        }
    });
}, { threshold: 0.1 });
```

### Animation Timing:
```javascript
// Data-delay para cascata:
<div class="scroll-fade-up" data-delay="100">
    Elemento 1 aparece em 100ms
</div>
<div class="scroll-fade-up" data-delay="200">
    Elemento 2 aparece em 200ms
</div>
```

## 🎨 CUSTOMIZAÇÃO FÁCIL

### Mudar Cor Principal (Rosa):
```css
/* dark-theme.css */
:root {
    --accent-primary: #ff6b35;  /* Laranja */
    --accent-secondary: #ff8c42;
}
```

### Mudar Velocidade Animação:
```css
.scroll-fade-up.active {
    animation: fadeInUp 0.5s ease;  /* Era 0.8s */
}
```

### Mudar Delay de Cascata:
```javascript
// scroll-animations.js
data-delay="200"  /* Muda o tempo */
```
