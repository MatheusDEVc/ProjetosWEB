## 🚀 Como Usar

### 1. **Navegação**
- Clique em **"Home"** para ir à página inicial
- Clique em **"Masculino"** ou **"Feminino"** para ver produtos específicos
- Use os botões de filtro para refinar a busca por categoria

### 2. **Adicionar Produtos**
- Clique em **"Ver Detalhes"** para abrir a página do produto
- Veja a descrição completa, especificações e benefícios
- Escolha a quantidade e clique em **"Adicionar ao Carrinho"**
- O carrinho atualiza automaticamente

### 3. **Gerenciar o Carrinho**
- Clique em **"Carrinho"** no header para abrir seu carrinho
- Veja todos os itens adicionados
- Aumente ou diminua a quantidade de cada produto
- Remova itens clicando no ✕
- Aplique cupom promocional (códigos disponíveis: **PROMO10**, **DESCONTO20**)

### 4. **Fazer Checkout**
- Clique em **"Ir para Checkout"**
- Preencha seus dados pessoais
- Preencha o endereço de entrega
- Escolha o método de pagamento (Cartão de Crédito, Débito, PIX ou Boleto)
- Revise o resumo do pedido
- Clique em **"Confirmar Pedido"**
- Aguarde a simulação de pagamento (2 segundos)
- Receba confirmação com número do pedido

### 5. **Acompanhar Pedidos** ⭐ NOVO!
- Clique em **"Meus Pedidos"** na navegação
- Veja todos os seus pedidos anteriores
- Filtre por status (Confirmado, Em Processamento, Preparando, Entrega, Entregue)
- Clique em **"Ver Rastreamento"** para detalhes completos
- Acompanhe o status em tempo real com timeline interativa
- Visualize a data estimada de entrega
- Veja detalhes de endereço e produtos

## 💳 Métodos de Pagamento Disponíveis

1. **Cartão de Crédito** - Com opção de parcelamento (1x, 2x, 3x ou 6x)
2. **Cartão de Débito** - Desconto automático da conta
3. **PIX** - Código QR exibido após confirmação
4. **Boleto** - Código gerado para pagamento

## 🎁 Cupons Promocionais

### Códigos Disponíveis:
- **PROMO10** - Desconto de R$ 50
- **DESCONTO20** - Desconto de R$ 100

Para usar: No carrinho, preencha o código no campo "Cupom Promocional" e clique em "Aplicar".
