## 🎬 Animações Implementadas

### 1. **SCROLL ANIMATIONS** (Intersection Observer)

Quando você scroll e elemento aparece na tela:

```
scroll-fade-up         → Elemento sobe com fade in
scroll-fade-down       → Elemento desce com fade in
scroll-slide-left      → Desliza da esquerda
scroll-slide-right     → Desliza da direita
scroll-scale           → Amplia com zoom
scroll-rotate          → Rotaciona entrando
```

### 2. **HOVER EFFECTS** (Ao passar o mouse)

```
Botões        → Levanta 5px com glow rosa
Cards         → Move para cima com sombra
Ícones        → Flutuam e expandem
Links         → Linha animada aparece embaixo
```

### 3. **SCROLL EFFECTS** (Ao fazer scroll)

```
Parallax      → Hero se move suavemente
Progress Bar  → Barra rosa mostra quanto scrollou
Mouse Follow  → Leve 3D nos cartões
```

### 4. **EFEITOS CONTÍNUOS**

```
glow-effect    → Brilho pulsante infinito
pulse-effect   → Pulso suave constante
float-effect   → Flutuação suave
```

## 🔧 Customizações Possíveis

### Mudar Cores Principais:
No `dark-theme.css`, edite:
```css
:root {
    --accent-primary: #d946a6;  /* Muda rosa principal */
    --bg-primary: #0f0f1e;      /* Muda fundo */
}
```

### Mudar Velocidade Animações:
No `dark-theme.css`:
```css
.scroll-fade-up.active {
    animation: fadeInUp 0.8s ease;  /* Muda 0.8s */
}
```

### Mudar Tempo de Delay:
No HTML ou via JS:
```html
<div class="category-card" data-delay="100">
    <!-- data-delay em ms -->
</div>
```
