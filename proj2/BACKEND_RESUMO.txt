🔌 API ENDPOINTS (Todos Prontos)

AUTENTICAÇÃO:
  POST   /api/auth/register     - Registrar usuário

PRODUTOS:
  GET    /api/products          - Listar todos os produtos

  POST   /api/orders            - Criar novo pedido
  GET    /api/orders            - Listar meus pedidos

ADMIN:
  GET    /api/admin/orders      - Todos os pedidos
  PUT    /api/admin/orders/:id/status - Atualizar status

TESTE:
  GET    /api/health            - Verificar se está online

❌ PROBLEMAS COMUNS & SOLUÇÕES
═══════════════════════════════

Erro 1: "npm: comando não encontrado"
├─ Causa: Node.js não instalado
└─ Solução: Baixe em https://nodejs.org

Erro 2: "EADDRINUSE: port 5000 already in use"
├─ Causa: Outra aplicação usando porta 5000
└─ Solução: Mude PORT em .env para 5001

Erro 3: "Cannot find module 'express'"
├─ Causa: Dependências não instaladas
└─ Solução: Execute setup.bat novamente

Erro 4: "ENOENT: no such file or directory"
├─ Causa: Pasta ou arquivo faltando
└─ Solução: Certifique-se que está em backend/

Erro 5: "No database found"
├─ Causa: database.db não criado ainda
└─ Solução: Aguarde 5 segundos e recarregue
