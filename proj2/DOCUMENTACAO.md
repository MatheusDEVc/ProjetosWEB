# 🎉 GlowUp Store - Projeto Concluído com Sucesso!

## ✅ Resumo do Projeto

Criei um **e-commerce profissional e totalmente funcional** para venda de produtos de saúde e beleza chamado **GlowUp Store**. O site está **100% pronto para o cliente visualizar e testar**.

## 🚀 Como Usar o Site

### Acesso:
1. Abra a pasta `proj2`
2. Clique com o botão direito em `index.html`
3. Selecione "Abrir com" e escolha seu navegador

### Fluxo de Compra:
1. **Home** → Explore as categorias
2. **Masculino/Feminino** → Veja produtos específicos
3. **Filtro** → Busque pela categoria desejada
4. **Ver Detalhes** → Conheça o produto
5. **Adicionar ao Carrinho** → Com quantidade
6. **Carrinho** → Revise e aplique cupom
7. **Checkout** → Preencha dados
8. **Pagamento** → Escolha método
9. **Confirmação** → Receba número do pedido
10. **Meus Pedidos** → Acompanhe seu histórico ⭐ NOVO!
11. **Rastreamento** → Veja status com timeline ⭐ NOVO!
